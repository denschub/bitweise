Bit & Byte

Ein Bit (binary digit, dt. Binärziffer) ist die kleinste elektronische Speichereinheit und gehört zum Binärsystem.

Das Binärsystem

Das binären Zahlensystem ist das bekannteste Dualsystem zur Darstellung von Zahlen. Es verwendet nur die Zahlen 0 und 1 zum Darstellen von Daten. Im Gegensatz dazu steht das Dezimalsystem welches die Zahlen 0 bis 9 verwendet.

Der Grund für die Verwendung des Binärsystems in der Elektrotechnik liegt nahe, denn auch dort gibt zwei eindeutige Zustände: Strom an, Strom aus.

Die Basis des Binärsystems ist 2. Der Stellenwert verdoppelt sich also bei jeder Stelle. Zur Verdeutlichung die Zahlen 0 bis 9: 

|---------|---------|
| Dezimal |  Binär  |
|---------|---------|
|       0 |       0 |
|       1 |       1 |
|       2 |      10 |
|       3 |      11 |
|       4 |     100 |
|       5 |     101 |
|       6 |     110 |
|       7 |     111 |
|       8 |    1000 |
|       9 |    1001 |
|---------|---------|

Das Byte

Ein Byte ist die kleinste addressierbare Datenmenge und besteht aus mehreren Bits. In der Regel ist ein Byte 8 Bits groß, kann aber auf bestimmten Rechnerarchitekturen variieren. Historisch gesehen war ein Byte die Anzahl der Bits zur Addressierung eines Zeichens.
