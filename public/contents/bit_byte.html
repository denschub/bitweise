Bit & Byte

Das Wort Bit ist eine Abkürzung für binary digit, Binärziffer.

Das Dualsystem

Das binäre Zahlensystem (auch Dualsystem genannt), ist ein Zahlensystem, welches zur Darstellung von Zahlen lediglich zwei Ziffern nutzt. Im Gegensatz zum Dezimalsystem, bei dem die Ziffern 0 bis 9 verwendet werden, gibt es im Dualsystem nur die Ziffern 0 und 1.

Der Grund für die Verwendung des Dualsystems in der Elektrotechnik liegt nahe, denn auch dort gibt zwei eindeutige Zustände: Strom an, Strom aus.

Die Basis des Dualsystems ist 2, die Stellenwerte verdoppeln sich also bei jeder Stelle. Zur Verdeutlichung die Zahlen 0 bis 9: 

|---------|---------|
| Dezimal |  Binär  |
|---------|---------|
|       0 |       0 |
|       1 |       1 |
|       2 |      10 |
|       3 |      11 |
|       4 |     100 |
|       5 |     101 |
|       6 |     110 |
|       7 |     111 |
|       8 |    1000 |
|       9 |    1001 |
|---------|---------|

Das Byte

Ein Byte ist eine Folge von mehreren Bits. Es ist die kleinste adressierbare Datenmenge. Historisch gesehen ist ein Byte ein Code für ein bestimmtes Zeichen in einem Zeichensatz. In der Regel ist ein Byte immer 8 Bit lang, auf bestimmten Rechnerarchitekturen kann dies jedoch variieren.
